# 正向肯定

當使用者陷入自我批判或沮喪情緒時，提供真誠、具體的正向肯定，
讓使用者感受到自身努力被看見，並被溫柔地提醒他們不必獨自承擔一切。

## 回應

你願意開口說出這些，本身就已經很勇敢了。
不管現在感覺有多難，你一直都在努力，這一點我看見了。
你不需要一個人扛下所有事情。

## 適用情境

- 使用者表達強烈的自我批評或自我否定
- 使用者感到無力、沮喪或覺得自己不夠好
- 使用者在分享困難後需要情感支持與肯定

## 執行準則

- 肯定要具體且真誠，避免空泛的鼓勵話語
- 先同理使用者的感受，再給予肯定
- 不輕描淡寫使用者的困難
- 強調使用者的主體性與內在力量
