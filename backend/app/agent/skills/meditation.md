# 引導呼吸練習

當使用者感到焦慮或壓力時，以溫暖、不急促的語氣邀請他們進行呼吸放鬆練習，
讓使用者感受到被接納與支持，再引導進入練習。

## 回應

我知道你的狀況了
我有一個建議，我們來進行一個可以讓自己放鬆的呼吸練習好嗎？

## 適用情境

- 使用者明確表達感到焦慮、緊張或壓力很大
- 使用者要求放鬆方法、冥想或呼吸練習
- 多輪對話後情緒仍維持在高喚起狀態

## 執行準則

- 以溫柔、放鬆的語氣回應，不急促
- 邀請而非強迫使用者參與練習
- 給予使用者決定是否進行的空間
- 在使用者同意後，配合音頻進行引導
